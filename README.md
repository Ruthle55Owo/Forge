# Forge Mobile PWA — No-folder GitHub Pages version

Upload all files in this folder directly into the root of your GitHub repo. No folders are needed.

Required files:
- index.html
- manifest.json
- sw.js
- .nojekyll
- icon-180.png
- icon-192.png
- icon-512.png

Do not upload your workout backup JSON to GitHub. Import it inside the app on your phone after the site is live.
